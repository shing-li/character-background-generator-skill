### Example Prompt
```

- 角色用途：
```

Notes
This skill is for fictional character creation only.

It should not be used to:

impersonate real people
create fraud identities
generate real credentials
create real passwords, phone numbers, addresses, ID numbers, or bank data
provide instructions for illegal activity
It can describe a fictional character’s attitude toward crime or gray-area behavior for storytelling purposes, but it should not provide operational instructions.
